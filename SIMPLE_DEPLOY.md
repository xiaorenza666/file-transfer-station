# 文件中转站 - 超简化部署指南

## 🚀 30秒快速部署

### 方式一：Linux/Mac（推荐）

```bash
# 1. 解压项目
unzip file-transfer-station.zip
cd file-transfer-station

# 2. 一键部署
bash deploy.sh
```

**就这么简单！** 应用会自动：
- ✓ 检查Node.js环境
- ✓ 安装所有依赖
- ✓ 配置数据库
- ✓ 启动服务器

然后打开浏览器访问：**http://localhost:3000**

## 🔧 常见问题

### Q: 提示"Node.js未安装"

**解决**：
1. 访问 https://nodejs.org
2. 下载 LTS 版本
3. 安装完成后重启电脑
4. 重新运行部署脚本

### Q: 提示"端口3000被占用"

**解决**：
1. 按 Ctrl+C 停止脚本
2. 运行：`PORT=3001 pnpm dev`
3. 访问：http://localhost:3001

### Q: 无法访问 http://localhost:3000

**解决**：
1. 确保脚本还在运行（没有按Ctrl+C）
2. 检查防火墙设置
3. 尝试访问 http://127.0.0.1:3000

### Q: 上传文件失败

**解决**：
1. 检查文件大小（最大50MB）
2. 检查网络连接
3. 查看浏览器控制台错误信息
